Welcome traveler! You've arrived at the Dendron workspace for CascadiaJS 2022! Dendron is the open source knowledge base for developers. All pages here are published from plaintext markdown and managed using git and GitHub.

You can fork the GitHub repo to get offline access to all the talks of CascadiaJS 2022. If you open this workspace using VSCode, you can use Dendron to navigate these notes as well as to add your own.

About Us

Dendron is a YC backed startup that wants to help humans organize information at scale. We are participating in the CascadiaJS Startup Fair - please drop by to say hi and pick up stickers and goodies!
